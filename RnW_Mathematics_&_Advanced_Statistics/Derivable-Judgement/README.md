# 📊 Health Data Analysis Project Summary

🏥 **Project Title:** Statistical Analysis of Health Records Dataset

### 🎯 Objective

The main goal of this project is to analyze health records of individuals and identify factors that may influence diseases such as **Diabetes** and **Hypertension** using statistical techniques.

### 📁 Dataset Overview

The dataset contains information about:

* 👤 Age & Age Group
* ⚖️ Weight
* 🚻 Gender
* 🌍 Region
* 🚬 Smoking Status
* 🏃 Exercise Frequency
* 📏 BMI (Body Mass Index)
* ❤️ Blood Pressure
* 🍬 Glucose Level
* 🩸 Cholesterol Level
* 🏥 Diabetes Status
* 💓 Hypertension Status
* 📅 Visit Date

### 🔍 Statistical Techniques Used

### 1️⃣ Hypothesis Testing

Formulated hypotheses to check whether:

* 🚬 Smoking affects Diabetes prevalence.
* 📊 BMI differs across Age Groups.

### 2️⃣ Confidence Interval

Calculated **95% Confidence Intervals** for:

* 👤 Age
* ⚖️ Weight

This helps estimate the likely range of true population values.

### 3️⃣ Critical Value & P-Value

Calculated critical values and p-values to determine whether the null hypothesis should be accepted or rejected.

### 4️⃣ T-Test

Compared the BMI of:

* 🚬 Smokers
* 🚭 Non-Smokers

to check whether the difference is statistically significant.

### 5️⃣ Chi-Square Test

Examined the relationship between:

* 🚬 Smoking Status
* 🍬 Diabetes

to determine whether they are associated.

### 6️⃣ ANOVA Test

Compared BMI across different:

* 👥 Age Groups

to check whether significant differences exist.

### 7️⃣ Covariance & Correlation

Analyzed the relationship between:

* 👤 Age
* 📏 BMI

to measure how strongly these variables move together.

### 8️⃣ Interpretation of Results

Based on p-values and critical values:

* ✅ Accepted or Rejected hypotheses.
* 📈 Identified significant health factors affecting diseases.

---

# 🏆 Conclusion

This project demonstrates how statistical methods can be used to analyze healthcare data and make data-driven decisions. By applying **Confidence Intervals, T-Test, Chi-Square Test, ANOVA, Covariance, and Correlation**, we can better understand the impact of lifestyle and demographic factors on disease occurrence. 📊✨

**Tools Used:** 🐍 Python, Pandas, NumPy, SciPy.
