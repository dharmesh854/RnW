🏠 Household Income Analysis Project

This project analyzes a dataset of 150+ households using Statistics and Data Analysis techniques. 📊

🔍 Project Includes
📂 Data classification (Numerical & Categorical)
📈 Mean, Median, Mode calculation
📉 Range, Variance, Standard Deviation & IQR
🔔 Normal Distribution analysis
📊 Histograms and Boxplots visualization
↔️ Skewness and Kurtosis calculation
🎓 Income comparison based on Education Level and Urban/Rural areas
🎯 Objective

To understand household income patterns and apply statistical concepts using Python, Excel, or Google Sheets. 🚀
