This project is about analyzing student performance and attendance using Power BI.

First, the data was cleaned and prepared. All datasets were loaded, relationships were created, and missing or incorrect values were fixed so the data becomes accurate and ready to use.

Then, important DAX measures were created, such as:

  - Percentage Score (Score ÷ Max Score)
  - Average score for each subject
  - Attendance percentage
  - Behavior count by type
  - Performance category (High, Medium, Low)

After that, an interactive dashboard was built to show insights clearly. It includes:

this dashboard contains 4 Pages:
  1. Exec Dashboard (Main Page)
  2. Performance Analysis
  3. Student Details
  4. Tooltip Page

Included Visuals:
  - Bar Chart & Column Chart
  - Line Chart
  - Donut & Pie Chart 
  - Table & Matrix
  - Cards & KPIs
  - Gauge Charts
  - Custom Buttons

Finally, interactivity features were added:

  - Filters (slicers) for Class, Section, Subject, and Term
  - Drill-through to see individual student details
  - Tooltips for extra insights
